【TWSE 策略回測系統 - 使用說明】

一、初次安裝
1. 請確認您收到的資料夾中有以下兩個檔案/目錄：
   - TWSE_Strategy_App.exe (應用程式)
   - data 資料夾 (內含 twse_data.db 資料庫)

2. 請將這兩個檔案放在「同一個資料夾」中 (例如建立一個 "StockApp" 資料夾)。
3. 直接點擊 TWSE_Strategy_App.exe 即可執行。

二、更新程式版本
如果您收到新版的 TWSE_Strategy_App.exe：
1. 請直接將新的 exe 檔案覆蓋舊的檔案。
2. 您的 data 資料夾 (以及裡面的 twse_data.db) 可以保留，這樣您原本下載的歷史股價資料就不會消失。
   (除非發布者特別說明需要替換資料庫)

三、常見問題
- 若無法開啟，請確認 data 資料夾是否跟 exe 在旁邊。
- 打開時會出現黑底視窗是正常的，請勿關閉，該視窗關閉後程式也會結束。
